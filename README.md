## Overview
This project demonstrates a CI pipeline using GitHub Actions for a Maven-based Java project.

## Features
- Automated build and test execution using Maven
- CI triggered on push and pull requests
- Dependency caching to improve build performance
- Test result validation through JUnit
- Basic use of environment variables

## Tech Stack
- Java 17
- Maven
- JUnit 5
- GitHub Actions

## CI Workflow
The pipeline automatically builds the project and runs unit tests on every push and pull request. This helps ensure code reliability and prevents broken changes from being merged.
